About Me

I am a 3rd year engineering student in CSE branch. I developed my interest in web development field during my summer internship in 2019. Since then I have been making projects and learning more about technologies and frameworks to increase my knowledge. This project is based on front-end web development(2019). This website is made using an interactive sticky header, a Parallax page that tracks mouse movements and creates a transition effect. A responsive grid layout is made to showcase the portfolio images with a slider to navigate through different pictures. It uses animation and is made responsive for all devices using HTML, CSS, Bootstrap and JavaScript to add interactivity to the web page.
